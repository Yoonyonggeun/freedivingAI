# Audio Assets for Training Timer

This directory should contain the following audio files:

1. **hold_start.mp3** - Played when hold phase starts (e.g., single beep)
2. **hold_end.mp3** - Played when hold phase ends (e.g., double beep)
3. **rest_start.mp3** - Played when rest phase starts (e.g., soft tone)
4. **training_complete.mp3** - Played when all rounds complete (e.g., success chime)
